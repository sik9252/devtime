# Typography

디자인 시스템의 타이포그래피 스타일입니다.

## Font Family

Pretendard를 기본 폰트로 사용합니다.

## Typography Scale

각 단계별 폰트 스타일과 굵기 예시입니다.

### Heading (24px / 30px)

대제목에 사용되는 가장 큰 텍스트 스타일입니다.

<div className="flex flex-col gap-2 p-4 border rounded-lg bg-gray-50">
  <div className="text-heading-b">Heading Bold (700) - 다람쥐 헌 쳇바퀴</div>
  <div className="text-heading-s">Heading SemiBold (600) - 다람쥐 헌 쳇바퀴</div>
  <div className="text-heading-m">Heading Medium (500) - 다람쥐 헌 쳇바퀴</div>
  <div className="text-heading-r">Heading Regular (400) - 다람쥐 헌 쳇바퀴</div>
</div>
